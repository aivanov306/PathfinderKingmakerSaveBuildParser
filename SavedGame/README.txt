# Place your extracted player.json and party.json files here

To get your save files:
1. Navigate to: %USERPROFILE%\AppData\LocalLow\Owlcat Games\Pathfinder Kingmaker\Saved Games\
2. Find your save folder (e.g., Auto_1.zks, Manual_1.zks)
3. Extract the .zks archive using 7-Zip, WinRAR, or any ZIP-compatible archiver
4. Copy the extracted player.json and party.json files to this directory
